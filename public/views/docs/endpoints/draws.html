    drawS

    oscarspalk

    The drawS endpoints returns an image in your preffered format with a string.

    # Parameters

    The table below contains a list of all query types for this endpoint.

    Name Explanation Required Defaults to Usage
    string An array of chars true NULL ?string=YOUR INPUT
    backgroundColor A hexadecimal representation of a color for the background false 000000 ?backgroundColor=ff
    fontColor A hexadecimal representation of a color for the font false ffffff ?fontColor=000000
    fontSize An integer false 16 ?fontSize=20
    xPosition An integer representing the string's x-coordinate false 0 ?xPosition=120
    yPosition An integer representing the string's y-coordinate false 0 ?yPosition=120
    xSize An integer representing the returned image's x-size false 100 ?xSize=500
    ySize An integer representing the returned image's y-size false 100 ?ySize=500
    centered A boolean whether to center the text horizontally and vertically false false ?centered=true
    format An enumeration deciding which format the outputted image should have false jpg ?format=png

    CASE-SENSITIVE

    Our API is case-sensitive when it comes to parameters.
